# Forest Evolution Game - Version baseline 1.7.0 (Stable)

## 📌 版本摘要

此檔案記錄了 2026-02-01 定義的穩定基線版本 1.7.0。此版本已通過 8 人對戰模擬測試，包含所有核心遊戲機制，並優化了視覺體驗與管理員控制功能。

## 🛠️ 核心功能清單

1. **全自動對戰流轉 (Auto-Pilot)**: 支持無人值守的階段自動切換。
2. **AI 玩家系統**: 四種性格的 AI 玩家，具備智慧攻擊與競標決策。
3. **視覺特效 (VFX)**: 火、水、木、雷四屬性動態背景環境特效。
4. **管理員控制面板 (Admin)**: 支持即時監控、玩家操作與自定義技能池。
5. **競標系統**: 支持最高價顯示、自動延時與 HP 返還機制。
