# AdminPanel.vue 結構與修改規範

為確保開發溝通精確，本檔案定義了 `AdminPanel.vue` 的介面區塊名稱。
每次修改程式碼前，必須明確告知使用者正在修改哪一個區塊。

## 1. 遊戲列表頁 (Dashboard)

對應代碼：`v-if="viewMode === 'dashboard'"`

- **[列表區]**
  - 顯示遊戲代碼、人數、狀態。
  - 特徵：直書按鈕、兩欄資訊佈局。
- **[建立區]**
  - 輸入人數、建立房間。
  - 包含「書本 📖」設定按鈕。
- **[技能設定視窗]** (Modal)
  - 點擊書本後彈出。
  - 包含：全選功能、技能勾選列表。

---

## 2. 遊戲控制台頁 (Control Panel)

對應代碼：`v-else-if="viewMode === 'control'"`

- **[控制台-頂部資訊]**
  - 顯示回合數、階段、遊戲代碼、返回按鈕 (⬅)。
- **[控制台-流程控制]**
  - 按鈕群組：開始遊戲、進入攻擊/競標階段、強制結束。
- **[控制台-玩家管理]**
  - 玩家卡片列表。
  - 功能：調整 HP、顯示屬性、踢出玩家 (✖)。
- **[控制台-紀錄區]**
  - 最下方的 Log 捲動視窗。
- **[控制台-結算區]**
  - 遊戲結束後的排名顯示。
- **[控制台-彈窗]**
  - 踢人確認、強制結束確認等 Modal。

---

## 修改前宣告範例

> ⚠️ **準備修改位置：**
> **頁面**：[遊戲控制台頁] > **[玩家管理]**
> **內容**：調整玩家卡片的背景顏色。
